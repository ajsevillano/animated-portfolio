# Animated Portfolio

> A dummy photographer portfolio built to practice the Framer Motion animation library. Features scroll-triggered animations, page transitions, and a responsive layout built with styled-components.

![React](https://img.shields.io/badge/React-61DAFB?style=flat&logo=react&logoColor=black)
![Framer Motion](https://img.shields.io/badge/Framer_Motion-0055FF?style=flat&logo=framer&logoColor=white)
![styled-components](https://img.shields.io/badge/styled--components-DB7093?style=flat&logo=styledcomponents&logoColor=white)

## Features

- Page transition animations with Framer Motion
- Scroll-triggered animations via react-intersection-observer
- Multi-page layout with React Router v5
- Responsive design with styled-components

## Getting Started

```bash
npm install
npm start
```

Open `http://localhost:3000` in your browser.

## Tech Stack

- React 17
- Framer Motion v4
- styled-components v5
- react-intersection-observer
- React Router v5
